PremiumPhotography

PremiumPhotography is an elegant website showcasing the photographer's work, allowing potential clients to get familiar with his portfolio and get in touch through the site. The project was developed using Vue.js, HTML, and CSS, with a focus on a simple yet sophisticated design that ensures a high-quality user experience.

Sentic Apartments

This website is designed to provide potential clients with a comprehensive overview of the apartments and their amenities, with an easy way to get in touch for inquiries and bookings. Featuring a clean and elegant design, it offers all the necessary information about the services, allowing users to get a true sense of the accommodation’s quality. It is built using HTML, Vue.js, CSS, and utilizes Vue plugins for added functionality and interactivity.

Brajic

The Brajić Law Firm website is designed to provide clients with a comprehensive overview of the services offered, as well as essential information about the team and office activities. The site aims to inform potential clients about the legal services provided by the firm, offering easy access to contact details and additional information. It is built using HTML, CSS, and JavaScript for a seamless user experience.

Flesh studio

The Flesh Photography Studio website is designed to showcase the unique work of a photographer specializing in landscape photography, with a focus on the natural beauty of Croatia’s Dalmatia region. Through carefully selected images, the site allows users to explore the photographer's work, highlighting the beauty of the landscapes through the photographer's lens. The aim of the website is to present his portfolio, featuring a simple yet unique design that reflects the essence of the photographs. The site is built using HTML, CSS, and JavaScript.

BizzNews

BizzNews is a dynamic website showcasing the latest news from global portals. With continuous updates through an API, the site offers the most recent information from various sources, keeping users up to date with world events. Focused on dynamism, the website is built using Vue.js, along with HTML and CSS for structure and design. BizzNews delivers fast, up-to-date news in an easy-to-navigate format, providing users with fresh information at all times.

Branch Café

Branch Café is an elegant website providing all essential information about a beloved café, including a detailed menu, an introduction to the team, reservation options, and easy contact with the café. Additionally, the site keeps visitors up to date on events hosted at the café. Built with HTML, CSS, and JavaScript, the website ensures smooth navigation and offers users a seamless experience while providing all the necessary details about the café’s services and events.
